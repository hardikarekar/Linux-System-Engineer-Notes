#linux 
* `grep` command is used to search text or search inside files for lines that match a pattern.
* Basic syntax
	* `grep [options] pattern [file...]`
		* `pattern` 
			* What you want to search for.
		* `file`
			* File to search in.
* Example
	* `grep cat cat.txt`
* Ignore case `-i`
	* `grep -i CAT cat.txt`
