#linux 
* Both are pagers used to view long text files one screen at a time in the terminal.
* `less`
	* Open a file with `less`
		* `less file`
	* Search within file. Type inside `less`
		* `/keyword`
	* Show line numbers
		* `less -N file`
* `more`
	* Open a file with `more`
		* `more file`
	* Search within file
